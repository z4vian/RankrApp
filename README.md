# Welcome to your Rankr app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Rankr is a ranking app used to rank different types of media into separate lists. The app makes the user select between two different objects within the same class of media, and will generate a list of objects ranked 1-10.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

notes: 

used last.fm for music api 
used rawg for game database api 
used MDB for movie database api
